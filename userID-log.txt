    Updated Database to include 4 different tables
    (user, idea, comment, like) added new prepared statments to retrieve
    all that information. Added new functions to admin app.
    Can now:
        T: Create all tables
        D: Delete all tables
        1: Query for specific row(user, idea, comment)
            Asks the admin what type of row they want, the corresponding ids,
            and then displays the data with a few options
                For user:
                    - Update information(name, email, gen, sex, note)
                For idea:
                    - Upvote
                        if no like exists creates new like in database with userid, ideaid, and commentid
                        if like status is 0 or -1 it increment likes and set status to 1
                        if like status is 1 it will decrement likes and set status to 0
                    - unlike
                        if no like exists creates new like in database with userid, ideaid, and commentid
                        if like status is 0 or 1 it decrement likes and set status to -1
                        if like status is -1 it will increment likes and set status to 0
                (comments works the same way as idea except the commentid is actually set and not *NULL*)
        V: Query for all valid rows(user, idea, comment)
            Asks the admin what type of row they want then displays all
            rows and little data, ONLY valid rows
        *: Query for all rows(user, idea, comment)
            Asks the admin what type of row they want then displays all
            rows and little data, includes invalid rows as well
        -: Delete Row(user, idea, comment)
        +: Insert Row(user, idea, comment)
        ~: Update Row(user, idea, comment)
        /: Toggle Valid(user, idea, comment)
            Asks what type, and then the ids. Sets valid = !valid [true -> false] or [false -> true]
                If user:
                    toggles all users, ideas and comments with userid
                If idea:
                    toggles all ideas and comments with ideaid
                If comment:
                    toggles all comments with ideaid and commentid
        q: quit :T
    
    I think that's everything . . .
